XBNPE: TextPad built-in command (TPKC-project)

The TextPad built-in command (TPKC) project is how PhraseExpress communicates with TextPad. Using the TPKC project gives your PhraseExpress code direct access to its more than three hundred "built in" commands (including file manipulation and search and replace).

The TPKC-project must be installed (as part of the overall XBN-PhraseExpress installation), and then "built". The TPKC user-build-process takes more than an hour, but it requires minimal user-interaction, and is a largely one-time-only process--and I can run it for you!. (Also, if you just installed TextPad and have no custom key-commands, then you do not need to run the build at all.) Aside from the XBNPE-key-presses (XKP) project, no other XBN-PhraseExpress project can function until the TPKC user-build-process is complete.

top     [PREV / NEXT]   The TPKC user-build process: Step-by-step

Requirements: This section describes the final step (number 3 in the list) of the overall XBN-PhraseExpress installation process. This section assumes that you have never run the TPKC-build process before--or you wish to do a full-refresh of your TextPad key-command configuration.

If your TPKC-user-dir name is "tp_frsh_instl", then you do not need to run this process at all. It has already been done for you.)

Note to TPKC-admins: This documentation section is for TPKC-users only, who execute the TPKC sub-build-processes 10 through 15. Build processes 01 through 09 are for TPKC-administrators only--these first nine (really, ten) processes are completely built before XBN-PhraseExpress is ever downloaded by a user. There is additional technical information.

BACK     [PREV -- NEXT]     The TPKC user-build process: Step-by-step

The first step is to print, validate, and then save your "TextPad all-command list".

Then, in the TPKC-user-menu, execute

TextPad built-in command (TPKC) project >
   z-User-build processes >
      TPKC user-build process ONE of 2

This creates the user-specific command-configuration, which must imported to its required location. Overwrite (delete) any existing folders when importing.

This process takes about 45 minutes. There is minimal interaction required. Do not touch your computer during this time.

Once complete, import all phrase-files (there is one phrase-file per TextPad command-category) to their required location. Overwrite (delete) any existing folders when importing.

[top]   XBNPE-TPKC: Required TextPad-bootstraps

The TPKC project--and therefore, excepting the XKP project, the entirety of XBN-PhraseExpress--requires the below TextPad-bootstraps to be installed before the TPKC-project itself can be installed. This list is in addition to the tp-bootstraps for the XKP project.

  • Regex for marking all lines in the TextPad all-command list:
        ^(EditCopyTaggedLines|EditCutTaggedLines|BookmarkNext|BookmarkToggle|SearchFiles|FileManage|FileOpen|FileQuickOpen|SearchFind|SearchFindNext|SearchGoto|Preferences|SearchReplace)\b
  • Testing functions (accessible via xhtk)
    • TPKC only:
      • Command: zTestAllTpkcTextPadBootstraps
      • Action: testtpbstpkc
    • XKP and then TPKC:
      • Command: zTestAllXkpTpkcTPBootstraps
      • Action: testtpbsxkptpkc
  • Phrase-files:
    • As used by a TextPad "fresh install": main, non-main
    • Blank template: main, non-main

Phrase name Built-in command name Category Description
zTPBookmarkCopyAll EditCopyTaggedLines Edit Copies all lines currently tagged with a bookmark.
zTPBookmarkCutAll EditCutTaggedLines Edit Cuts all lines currently tagged with a bookmark.
zTPBookmarkGTNext BookmarkNext Search Goes to the next bookmarked line, if any.
zTPBookmarkToggle BookmarkToggle Search Marks the currently-UN-marked line, or un-marks the currently-marked line.
zTPFFindDlgOpen SearchFiles Search Opens the "Find In Files" dialog.
zTPFileOpen FileOpen File Opens the "Open file(s)" dialog.
zTPFindDlgOpen SearchFind Search Opens the "Find" dialog
zTPFileManager FileManage File Opens the "Manage Files" dialog.
zTPFileQOpen FileQuickOpen File Opens the "Quick Open File" dialog.
zTPFindNext SearchFindNext Search With the find-dialog closed, finds the next search-term down.
zTPGTDlgOpen SearchGoto Search Opens the "Go To" dialog.
zTPPrefsDlgOpen Preferences Configure Opens the "Preferences" dialog.
zTPRplcDlgOpen SearchReplace Search Opens the "Replace" dialog
