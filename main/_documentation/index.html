XBN-PhraseExpress is a large set of generically useful MS Windows scripting tools, written in PhraseExpress and TextPad. TextPad is the plain-text editor written by Helios Software Solutions. XBNPE is significantly for creating phrases in TextPad--and therefore most phrases are logically only-callable in TextPad. There are, however, many useful features for those developing phrases in any environment, including the PhraseExpress application itself.

There is a significant setup necessary in order for XBN-PhraseExpress and TextPad to communicate properly, but it is largely a one-time-only process.

[BACK]   XBN-PhraseExpress: Installation

Install TextPad. XBNPE is verified with TextPad version 7.0.9. It should work with all 7.* versions. It will not work with earlier versions.

Install PhraseExpress. XBNPE was built with "PhraseExpress Standard v9.1.41s". It may work with other 9.* versions of PhraseExpress. It will not work with earlier versions.

Download and install XBN-PhraseExpress.

Configure global directories:

[top]   XBN-PhraseExpress: global configuration

Global configuration variables that must be correct, in order for anything in XBN-PhraseExpress to function.

Variable Description and requirements
zzTextPadWindowName The name of the TextPad application's window title, as required by {#FOCUS ...}. May include asterisks (such as "*TextPad*"). Used for re-focusing on the main TextPad document window, when opening and closing both TextPad and PhraseExpress dialogs.
zzCurrVersionOfPETextEditorWUndrscrs The current version of TextPad, using underscores instead of dots. For example, if, in
    TextPad > Help > About
it says version "7.0.9", then this value must be "7_0_9".

This is the postfix of the name of the version-specific directory in the XBNPE-TPKC project.
zzTPTxtDefaultWordWrapOnOrES Plain-text phrase-files are named with the postfix ".px.txt" and are therefore included in the "Text" document class. If the default word-wrap state for the text document class is
  • checked, then the value of this phrase must be "on"
  • unchecked, then this must be nothing (the empty-string: '').
(Screenshot below)
zzTPTxtDefaultStripTrailingWSOnOrES If the Text document class setting "Strip trailing spaces from lines when saving" is
  • checked, then the value of this phrase must be "on"
  • unchecked, then this must be nothing (the empty-string: '').
This refers to whitespace being at the end (right-side) of each line. Both tabs and spaces are stripped.

(Screenshot below)

Warning: Significant portions of XBNPE (particularly the larger build-processes) will fail unless "Word wrap text > Save with no breaks in lines" is selected.
