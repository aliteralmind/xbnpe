XBNPE Phrase-creator (PCC) project

A phrase-creator is one of the four main XBNPE "normal" phrase-types.

A "phrase creator" is an action that serves a single purpose: Printing a call to another phrase or to a standard PhraseExpress function. In addition to this project, also see the 'xh...' drop-down menu.

This documentation is by example. Most phrase-creators end by presenting the user with a 'BALLOON' containing further information and syntax. Note that this is the only phrase-type that (always) does this, since it is the only type that is intended for use directly by the user and executes right now.

[top]   XBNPE-PCC: Examples: Raw PhraseExpress key-presses

  • krk: A raw key-press, unmodified.
        {#[cursor]}
    Follow with the name of the key, such as 'TAB' or 'F1'
  • krkcs: 'krk', modified with 'CTRL' and 'SHIFT'.
        {#CTRL {#SHIFT {#[cursor]}}}
  • krha: Same as 'krka', for a specific and printable key.
        {#ALT -chars [cursor]}
  • kru: Same as 'krh', with the additional ' -count' parameter.
        {#NONE -chars [cursor] -count XXX}
  • krnl: Same as 'krkl', with the ' -count' parameter.
        {#LWIN {#[cursor] -count XXX}}
  • krxs: Same as 'krks', where nothing is modified.
        {#SHIFT [cursor]}
    This is intended for when the to-be-modified key is in the clipboard. Follow this by pasting it.

This list is not complete. Using any of these phrase-creators will present you with a complete syntax-summary.

[top]   XBNPE-PCC: Examples: The most fundamental phrase: {#insert ...}

The 'insert' tag is the most basic PhraseExpress function.

  • ins: An insert tag.
        {#insert [cursor]}
  • insb:
        {#insert [paste]}[cursor]
  • inwl: Call with the cursor immediately following a phrase-name
        {#insert [paste]}[cursor]
  • inwr: Call with the cursor immediately preceding a phrase-name
        {#insert [paste]}[cursor]
  • insx:
        {#insert XXX}[cursor]
  • insqs: For creating diagnostic text for MSGBOX-es or BALLOON-s.
        [paste]='{#insert [paste]}'[cursor]
  • inns: Note that only the outer insert is escaped
        {#insert {#insert [cursor]}}
  • innsb:
        {#insert {#insert [paste]}}[cursor]

This list is not complete. Using any of these phrase-creators will present you with a complete syntax-summary.

[top]   XBNPE-PCC: Examples: Raw PhraseExpress functions

  • ins: An insert tag.
        {#insert [cursor]}
  • bali:
        {#BALLOON -title [cursor] -text XXX}
  • lop:
        {#LOOP [cursor] -count XXX}
  • cqdit:
        {#COND -if {#insert [paste]} [cursor] -then YYY -else YYY}
  • cilb:
        {#CALC [paste] [cursor] -round 0 -thousands none}
  • ifl:
        {#INSERTFILE [cursor]}
  • gthcb:
        {#LENGTH {#INSERTCLIPBOARD}}[cursor]
  • rpl:
        {#REPLACE XXX -oldtext YYY -newtext YYY}[cursor]

This list is not complete. Using any of these phrase-creators will present you with a complete syntax-summary.

[top]   XBNPE-PCC: Examples: Escaped-ness

All phrase-creators in the PPC-project may be followed by a 'p' or 'q', for printing the phrases at a specific level of escaped-ness. Escaped-ness is critical when literally displaying PhraseExpress code, when creating phrases that create phrases, and also when storing PhraseExpress code into variables or the clipboard. There are up to ten levels of escaped-ness available.

Examples:

  • krhsp: Singly-escaped.
        \\\{#SHIFT -chars [cursor]\\\}
  • lopq: Doubly-escaped (provide '2' at the prompt).
        \\\\\\{#LOOP [cursor] -count XXX\\\\\\}
  • balwq: Triply-escaped (provide '3' at the prompt).
        {#insert zOYe2}#BALLOON -title Please wait... -text [cursor]{#insert zCYe2}
  • gthbq: Ten-levels-escaped (provide '10' at the prompt).
        {#insert zOYe9}#LENGTH [paste]{#insert zCYe9}[cursor]

From unescaped to way-escaped

  1. {#insert ...}
  2. \\\{#insert ...\\\}
  3. \\\\\\{#insert ...\\\\\\}
  4. {#insert zOYe2}#insert ...{#insert zCYe2}
  5. {#insert zOYe3}#insert ...{#insert zCYe3}
  6. {#insert zOYe4}#insert ...{#insert zCYe4}
  7. {#insert zOYe5}#insert ...{#insert zCYe5}
  8. {#insert zOYe6}#insert ...{#insert zCYe6}
  9. {#insert zOYe7}#insert ...{#insert zCYe7}
  10. {#insert zOYe8}#insert ...{#insert zCYe8}
  11. {#insert zOYe9}#insert ...{#insert zCYe9}
