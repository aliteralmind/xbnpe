XBN-PhraseExpress: Drop-down menus (DDMenu)

There are two main drop-down menus (heirarchical pop-up menus) made available by XBN-PhraseExpress:

  • [go]   The phrase-creator ddmenu ('xh...')
  • [go]   The XBN-PhraseExpress user-menu ('zCASpDNR_SLASH')

Drop-down menu phrases must be configured in a particular way.

[top]     The phrase-creator ddmenu ('xh...')

XBN-PhraseExpress provides a drop-down menu (a heirarchical pop-up menu) containing most available phrases in the XBN-PhraseExpress project. It is created and maintained with the Phrase-creator drop-down-menu utilities (PMDM).

By default, it is triggered by typing 'xh'. Clicking on most any phrase will print a call to it. This is therefore a phrase-creator ddmenu (also see the XBNPE Phrase-creator (PCC) project). Some items may provide extra functionality, such as printing a corresponding close-phrase (as demonstrated below, with 'xhsro6').

While most items in this xh-ddmenu print phrase-calls, there are exceptions. For example, all individual (and "master") build process in the TextPad Built-In Commands (TPKC) project (via 'xhtkab') are actually triggered when clicked on. Note that accessing the TPKC-build-processes in this manner (and any of the 'xhtk' phrases) are intended for TPKC-administrators only. TPKC users should only access their build processes via the XBN-PhraseExpress user-menu.

Demonstration: 'xhsro6'

Note these menus are configured to have a particular appearance and behavior.

Note that no flags are (un)set in the dialog-close phrase, regardless the flags that happen to be in the chosen open-phrase. For example:
    {#insert zRplcDlgOpenRE}{#insert zRplcFieldWith}{#insert zRplcDlgClose}
In this case, you'll need to manually add the RE, if desired:
    {#insert zRplcDlgOpenRE}{#insert zRplcFieldWith}{#insert zRplcDlgCloseRE}

[top]     The XBN-PhraseExpress user-menu ('zCASpDNR_SLASH')

Major functionality for XBNPE users is listed in the "XBNPE User Menu". By default, this is accessed by clicking on 'zCASpDNR_SLASH' (Ctrl+Alt+Shift+\).
