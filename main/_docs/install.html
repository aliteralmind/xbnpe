XBN-PhraseExpress installation: Step-by-step

This documentation describes the specific steps necessary for installing XBN-PhraseExpress, up-through the XKP and TPKC projects, including "building" the TPKC-project.

Notes:

  • XBN-PhraseExpress was developed and tested exclusively in:
        Microsoft Windows 7 Home Premium (32-bit)
        Version 6.1.7601 Service Pack 1 Build 7601
  • The PhraseExpress professional license is not required in order to use XBN-PhraseExpress
  • The final step in this process takes an hour, during which time you may not use your computer. I can do this step for you.
  • XBN-PhraseExpress is in beta. It is thoroughly tested on a single machine, in two Windows accounts--one with my own key-command setup, one with a fresh-installation of TextPad. Backup your TextPad settings before proceeding. Please help improve XBN-PhraseExpress by reporting any obvious problems in the issue tracker.

START

Install and configure TextPad

  • Recommended version: Download version 7.0.9 from textpad.com.
  • Minimum version required: Any version 7.*.*.

Open TextPad and properly configure the Text Document Class, as required by XBN-PhraseExpress. The specific settings you make here are needed in a future step.

It is recommended that

  • Your current key-command setup is backed up
  • The status-bar is displayed:
        TextPad > View > Status Bar
  • Line-numbers are displayed by default:
        TextPad > Configure > Preferences > View > Line numbers
  • It is required that search-results are not docked:
        TextPad > Configure > Preferences > View > Dockable log windows > Search Results
    This is strictly only necessary for the last step in this installation (the TPKC user-build-process), and much of the find-in-files functionality in the SNR-project.

Install PhraseExpress

  • Recommended version: Download version v9.1.45s from phraseexpress.com.
  • Minimum version required: XBN-PhraseExpress should work with all versions of PhraseExpress starting with v9.*.*.

It is recommended to configure drop-down-menu phrases (pop-up-heirarchies) in the same manner as presented throughout this documentation.

All XBNPE documentation expects that you are in PhraseExpress "Expert mode". If this button says "Easy mode", then you are indeed in expert-mode (the button is for switching back to easy mode). If it says "Expert mode", then click it.

Download XBN-PhraseExpress. Extract the download-archive into, for example:
    C:\programming\phrase_express\
such that the root of the newly-extracted directory is
    C:\programming\phrase_express\xbnpe\

Note that
    C:\programming\phrase_express\
is called the XBNPE "parent" directory, and
    C:\programming\phrase_express\xbnpe\
is called the XBNPE "main" directory.

These values are needed in a future step.

The source-code for XBN-PhraseExpress is now properly installed.

Open the PhraseExpress application by clicking on the red exclamation point (inside a white "speech bubble"), right-click on any item above "Create new phrase", then select "Edit element"

    Import > PhraseExpress phrase file
