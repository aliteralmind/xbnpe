XBN-PhraseExpress: Drop-down menus (DDMenu)

There are two main drop-down menus (heirarchical pop-up menus) made available by XBN-PhraseExpress:

  • [go]   The "xh..." phrase-creator ddmenu
  • [go]   The XBN-PhraseExpress user-menu

Drop-down menu phrases must be configured in a particular way.

[top]     The phrase-creator ddmenu ("xh...")

XBN-PhraseExpress provides a drop-down menu (a heirarchical pop-up menu) containing most available phrases in the XBN-PhraseExpress project. It is created and maintained with the Phrase-creator drop-down-menu utilities (PMDM).

By default, it is triggered by typing "xh". Clicking on most any phrase will print a call to it. This is therefore a phrase-creator ddmenu (also see the XBNPE Phrase-creator (PCC) project). Some items may provide extra functionality, such as printing a corresponding close-phrase (as demonstrated below, with "xhsro6").

While most items in this xh-ddmenu print phrase-calls, there are exceptions. For example, all individual (and "master") build process in the TextPad Built-In Commands (TPKC) project (via "xhtkab") are actually triggered when clicked on. Note that accessing the TPKC-build-processes in this manner (and any of the "xhtk" phrases) are intended for TPKC-administrators only. TPKC users should only access their build processes via the XBN-PhraseExpress user-menu.

Demonstration: "xhsro6"

Note these menus are configured to have a particular appearance and behavior.

[top]     The XBN-PhraseExpress user-menu

Major functionality for XBNPE users is listed in the "XBNPE User Menu".

Default key-command

The default key-command for the XBNPE user-menu is
    zCASpDNR_SLASH

The XBNPE user-menu is the only key-command assignment outside of the XBNPE-Default key-commands folder. This one key-command is assigned to the entire folder (the phrase-file itself, after importing):

  • On disk (phrase-files)
        {#insert zROOT_DIR_XBNPE}\z__xbnpe_user_menu__forHotKeys_only.px.txt
  • PhraseExpress folder (to import to) 
    XBNPE: Parent folder >
   z-XBNPE: For HotKeys only

Consequence of consolidating all XBNPE-user-menu key-command phrases

Normally, these phrases would be in their logical locations throughout the XBNPE folder-heirarchy. For example,
    TextPad built-in commands (TPKC) project
would reside in

    XBNPE: Parent folder >
       XBN-PhraseExpress >
(go)      XBNPE-TPKC: TextPad built-in commands >
             XBNPE-TPKC: User-spcfc phrases: [TPKC_USER_DIR_NAME] >
                z-DDMENU of all active TP-built-ins (AG:15)

Putting it there would allow you to navigate the entire menu with the arrow-keys, but it would also require changing the key-command in all these locations. The consequence of putting them all into this one phrase-file, is that one of the four choices must be made by manually selecting it (via the enter-key) before any sub-choices are displayed.
