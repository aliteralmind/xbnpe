XBNPE Phrase-creator (XPC) project

A phrase-creator is one of the four "normal" phrase-types.

A "phrase creator" is an action that serves a single purpose: Printing a call to another phrase, or to a standard PhraseExpress function. (See: "xh..." phrase-creator drop-down menu)

This documentation is by example. Most phrase-creators end by presenting the user with a "BALLOON" containing further information and syntax.

[top]   XBNPE-XPC: Examples: Raw PhraseExpress key-presses

(See: "Raw" PhraseExpress key-presses)

This list is not complete. Using any of these phrase-creators will present you with a complete syntax-summary.

  • krk: A raw key-press, unmodified.
        {#[cursor]}
    Follow with the name of the key, such as "TAB" or "F1"
  • krkcs: "krk", modified with "CTRL" and "SHIFT".
        {#CTRL {#SHIFT {#[cursor]}}}
  • krha: Same as "krka", for a specific and printable key.
        {#ALT -chars [cursor]}
  • kru: Same as "krh", with the " -count" parameter.
        {#NONE -chars [cursor] -count XXX}
  • krnl: Same as "krkl", with the " -count" parameter.
        {#LWIN {#[cursor] -count XXX}}
  • krxs: Same as "krks", where nothing is modified.
        {#SHIFT [cursor]}
    This is for when the to-be-modified key is in the clipboard. Follow this by pasting it.

[top]   XBNPE-XPC: Examples: The most fundamental phrase: {#insert ...}

The "insert" tag is the most basic PhraseExpress function.

This list is not complete. Using any of these phrase-creators will present you with a complete syntax-summary.

  • ins: An insert tag.
        {#insert [cursor]}
  • inb:
        {#insert [paste]}[cursor]
  • inl: Call with the cursor immediately following a phrase-name
        {#insert [paste]}[cursor]
  • inr: Call with the cursor immediately preceding a phrase-name
        {#insert [paste]}[cursor]
  • inx:
        {#insert XXX}[cursor]
  • insqs: For creating diagnostic text for MSGBOX-es or BALLOON-s.
        [paste]='{#insert [paste]}'[cursor]
  • inn: Note that only the outer insert is escaped
        {#insert {#insert [cursor]}}
  • innb:
        {#insert {#insert [paste]}}[cursor]

inlq:

Transforms this

zPhraseName1[cursor]
zPhraseName2
zPhraseName3
zPhraseName4
zPhraseName5
zPhraseName6
zPhraseName7

To this

{#insert zPhraseName1}{#insert zPhraseName2}{#insert zPhraseName3}{#insert zPhraseName4}{#insert zPhraseName5}{#insert zPhraseName6}{#insert zPhraseName7}[cursor]

This list is not complete. Using any of these phrase-creators will present you with a complete syntax-summary.

  • ins: An insert tag.
        {#insert [cursor]}
  • bali:
        {#BALLOON -title [cursor] -text XXX}
  • lop:
        {#LOOP [cursor] -count XXX}
  • cqdit:
        {#COND -if {#insert [paste]} [cursor] -then XXX -else XXX}
  • cilb:
        {#CALC [paste] [cursor] -round 0 -thousands none}
  • ifl:
        {#INSERTFILE [cursor]}
  • gthcb:
        {#LENGTH {#INSERTCLIPBOARD}}[cursor]
  • rpl:
        {#REPLACE XXX -oldtext YYY -newtext YYY}[cursor]

A listing of all "main" XPC phrase-creators can be obtained via 'xhpcp'. These "main" phrase-creators, when used, pop-up a "BALLOON" containing information and a complete syntax-summary on all related phrases.

[top]   XBNPE-XPC: Escaped-ness

PhraseExpress code is escaped by inserting three down-right-slashes ("\\\") before all open and close curlys:
    \\\{
    \\\}
Unescaping is done by simply removing the three slashes immediately preceeding the curly.

All phrase-creators in the XPC-project may be followed by a 'p' or 'q', for printing the phrases at a specific level of escaped-ness. Escaped-ness is critical when literally displaying PhraseExpress code, when creating phrases that create phrases, and also when storing PhraseExpress code into variables or the clipboard. There are up to ten levels of escaped-ness available.

Examples:

  • krhsp: Singly-escaped.
        \\\{#SHIFT -chars [cursor]\\\}
  • lopq: Doubly-escaped (provide '2' at the prompt).
        \\\\\\{#LOOP [cursor] -count XXX\\\\\\}
  • balwq: Triply-escaped (provide '3' at the prompt).
        {#insert zOYe2}#BALLOON -title Please wait... -text [cursor]{#insert zCYe2}
  • gthbq: Ten-levels-escaped (provide "10" at the prompt).
        {#insert zOYe9}#LENGTH [paste]{#insert zCYe9}[cursor]

From unescaped to way-escaped

  1. {#insert ...}
  2. \\\{#insert ...\\\}
  3. \\\\\\{#insert ...\\\\\\}
  4. {#insert zOYe2}#insert ...{#insert zCYe2}
  5. {#insert zOYe3}#insert ...{#insert zCYe3}
  6. {#insert zOYe4}#insert ...{#insert zCYe4}
  7. {#insert zOYe5}#insert ...{#insert zCYe5}
  8. {#insert zOYe6}#insert ...{#insert zCYe6}
  9. {#insert zOYe7}#insert ...{#insert zCYe7}
  10. {#insert zOYe8}#insert ...{#insert zCYe8}
  11. {#insert zOYe9}#insert ...{#insert zCYe9}
