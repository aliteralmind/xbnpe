[top]     65/10: Quotient-only

Basic division. Remainder is ignored.

dq1quotient
(*Input:*)
            {#SETPHRASE -description 
   zvDIVIDEND_TOP -content 
   65 
            -autotext zvDIVIDEND_TOP -folder ctvv_folder}{#SETPHRASE -description 
   zvDIVISOR_BTM -content 
   10 
            -autotext zvDIVISOR_BTM -folder ctvv_folder}
(*Go:*)
   {#insert zQuotient_M_divisorDividend}
(*Output:*)
   {#insert zvDIVIDEND_TOP}/{#insert zvDIVISOR_BTM}={#insert zvQUOTIENT}

Output: 65/10=6

[top]     65/10: Quotient, modulus and 1-if-mod-flag

Division with remainder, and an "if-remainder" flag.

"zv1_IF_MOD_GT0" is a flag representing if there is a remainder. It is set to

  • If there is a remainder: 1
  • If there is no remainder: 0

dq3qmod1if
(*Input:*)
            {#SETPHRASE -description
   zvDIVIDEND_TOP -content 
   65 
            -autotext zvDIVIDEND_TOP -folder ctvv_folder}{#SETPHRASE -description 
   zvDIVISOR_BTM -content 
   10 
            -autotext zvDIVISOR_BTM -folder ctvv_folder}
(*Go:*)
   {#insert zQuotientModulo1IfMod_M_divisorDividend}
(*Output:*)
   {#insert zvDIVIDEND_TOP}/{#insert zvDIVISOR_BTM}={#insert zvQUOTIENT}, 
    remainder {#insert zvMODULO} (if-mod-flag={#insert zv1_IF_MOD_GT0})

Output: 65/10=6, remainder 5 (if-mod-flag=1)

[top]     65/10: Quotient, modulus, 1-if-mod-flag, and total-block-count

[Go to example 1, example 2]

"Total blocks" ("zvTTL_DIV_BLKS") is the quotient, plus

  • If there is a remainder: 1
  • If there is no remainder: 0
Specifically:
    zvQUOTIENT + zv1_IF_MOD_GT0

Total-block-count is used to process a set of things, handling them in groups of at most a certain size (as defined by the divisor), without neglecting those items in the remainder group.

[BACK]     65/10: Total-block-count: Example 1/2: Trivial

dq4qmod1ifblk
(*Input:*)
            {#SETPHRASE -description 
   zvDIVIDEND_TOP -content 
   65 
            -autotext zvDIVIDEND_TOP -folder ctvv_folder}{#SETPHRASE -description 
   zvDIVISOR_BTM -content 
   10 
            -autotext zvDIVISOR_BTM -folder ctvv_folder}
(*Go:*)
   {#insert zQuotientModulo1IfModTtlBlks_M_divisorDividend}
(*Output:*)
   {#insert zvDIVIDEND_TOP}/{#insert zvDIVISOR_BTM}={#insert zvQUOTIENT}, 
    remainder {#insert zvMODULO} (if-mod-flag={#insert zv1_IF_MOD_GT0}, 
    total-blocks={#insert zvTTL_DIV_BLKS})

Output: 65/10=6, remainder 5 (if-mod-flag=1, total-blocks=7)

[BACK]     65/10: Total-block-count: Example 2/2: More realistic

dq5splitevengroups
(*Init:*)
            {#SETPHRASE -description
   zvSOME_XS -content 
   xxxxxxxxxxxxxxxxxxxxxx 
            -autotext zvSOME_XS -folder ctvv_folder}{#SETPHRASE -description 
   zvDIVIDEND_TOP -content 
   {#LENGTH {#insert zvSOME_XS}} 
            -autotext zvDIVIDEND_TOP -folder ctvv_folder}{#SETPHRASE -description 
   zvDIVISOR_BTM -content 
   5 
            -autotext zvDIVISOR_BTM -folder ctvv_folder}
(*Determine group count:*)
   {#insert zQuotientModulo1IfModTtlBlks_M_divisorDividend}
Dividing {#insert zvDIVIDEND_TOP} x-s into groups of, 
   AT MAXIMUM, {#insert zvDIVISOR_BTM}.{#ENTER}
   {#insert zvDIVIDEND_TOP}/{#insert zvDIVISOR_BTM}={#insert zvQUOTIENT}, 
   remainder {#insert zvMODULO} (if-mod-flag={#insert zv1_IF_MOD_GT0}, 
   total-blocks={#insert zvTTL_DIV_BLKS}){#ENTER -count 2}
(*Output:*)
   {#LOOP {#insert zzSplitNextXGroup_M_xsDvsr} -count {#insert zvTTL_DIV_BLKS}}

zzSplitNextXGroup_M_xsDvsr -- Needs zvSOME_XS, zvDIVISOR_BTM
(*Get current group to clipboard, update all-Xs var:*)
   {#SETCLIPBOARD 
      {#SUBSTR {#insert zvSOME_XS} 
         -from 0 
         -count 
            {#insert zvDIVISOR_BTM}}}
                  {#SETPHRASE -description 
   zvSOME_XS -content 
   {#SUBSTR {#insert zvSOME_XS} 
      -from {#CALC {#insert zvDIVISOR_BTM} + 1 -round 0 -thousands none} 
      -count {#CALC {#LENGTH {#insert zvSOME_XS}} - {#insert zvDIVISOR_BTM} -round 0 -thousands none}} 
                  -autotext zvSOME_XS -folder ctvv_folder}
(*Output:*)
   {#INSERTCLIPBOARD}{#ENTER}

Output:

Dividing 22 x-s into groups of, AT MAXIMUM, 5.
22/5=4, remainder 2 (if-mod-flag=1, total-blocks=5)
