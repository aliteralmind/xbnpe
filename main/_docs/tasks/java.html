XBNPE: Java

The XBNPE-Java project provides actions for creating and documenting non-GUI Java code. This documentation is by example only. Calling any of the below actions present a balloon containing a complete syntax summary of related actions.

Contents:

  • [go]   Basic auto-completes
  • [GO]   Function signature: return-type, name, parameters, entire
  • [go]   JavaDoc:
    • [GO]   Basic auto-completes
    • [GO]   Zero-parameter links
    • [GO]   Class-file links
    • [GO]   Internal object links
    • [GO]   Pre-link class-file links
    • [GO]   @param for parameters in below signature
